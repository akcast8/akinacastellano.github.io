Project Overview

Developed a mobile application in collaboration with an Emory University physician to promote melanoma awareness and support early skin cancer detection through image-based screening and educational resources. The app combines machine learning-powered analysis with guided clinical assessment to help users make informed health decisions.

Tech Stack

  • Programming Language: Swift
    • Used to implement the app's core features and user interface.
  • IDE & SDK: Xcode & iOS SDK
    • Apple's development environment and tools for building, testing, and deploying iOS apps.
  • Machine Learning: Apple Create ML
    • Developed and trained models for melanoma detection from skin images.
  • Backend & Data Storage: Firebase
    • Secured user-specific records, scan history, and image storage.
  • Project Management: Jira
    • Tracked tasks, milestones, and team collaboration throughout development.

Key Features

  • Machine learning-based skin image scanning with severity diagnosis
  • Image upload and scan history tracking
  • ABCDE melanoma assessment checklist
  • Educational resource hub
  • “Find Medical Professionals Nearby” feature
  • Secure Firebase storage

What I Did

  • Implemented core iOS features in Swift including ABCDE Checklist View
  • Integrated Apple Create ML models for image-based screening
  • Built Firebase-backed user data storage for scan records
  • Incorporated trusted clinical content
  • Collaborated with physician and team members using Jira
  • Presented project at Expo (3rd Place)
