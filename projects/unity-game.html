Project Overview

Developed a Unity-based 3D third person survival game emphasizing real-time character control, intelligent enemy behavior, and immersive environmental interaction. The project showcases core game development principles through polished mechanics, AI systems, and player-driven exploration.

Tech Stack

  • Game Engine: Unity - Used for 3D environment creation, game logic, physics, and rendering.
  • Programming Language: C# - Implemented player controls, AI behavior, and gameplay mechanics.
  • AI & Navigation: NavMesh - Created dynamic pathfinding and intelligent computer behavior for NPCs and enemies.
  • Animation: Animator with Root Motion - Developed smooth player and NPC animations for realistic movement.
  • Version Control: Git - Managed codebase, tracked changes, and collaborated with team members.
  • Project Management: Trello - Organized tasks, milestones, and team workflow.

Key Features

  • Fully animated third-person player controller using root motion
  • NavMesh-based enemy AI with dynamic pathfinding and chase behavior
  • Puzzle-solving mechanics integrated into level progression
  • Interactive UI menus for gameplay flow and player feedback
  • Immersive level design supporting exploration and survival gameplay

What I Did

  • Implemented gameplay mechanics and player controls using C# in Unity
  • Assisted with level design and environmental layout
  • Developed and tuned enemy AI with Unity NavMesh for real-time pursuit behavior
  • Built interactive UI elements and menus to enhance player experience
  • Collaborated with team members using Trello and version control
