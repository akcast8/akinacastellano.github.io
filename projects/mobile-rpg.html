Project Overview

Designed and developed a 2D RPG mobile game in Android Studio that features character selection, progressively challenging levels, and dynamic enemy encounters. The project demonstrates core mobile game development concepts including game state management, difficulty scaling, and UI integration.

Tech Stack

  • Programming Language: Java
    • Implemented core gameplay logic, character selection, and combat mechanics.
  • IDE & SDK: Android Studio & Android SDK
    • Developed, tested, and deployed the game for Android devices.
  • UI & Game State Management: Android Views and custom logic
    • Managed game levels, scoreboards, and player progression.
  • Project Management: Jira
    • Planned tasks, tracked progress, and coordinated team workflow.

Key Features

  • Character sprite selection system with distinct visual styles
  • Multi-level progression with increasing difficulty and varied texture tiles
  • Enemy combat mechanics featuring differing health levels and skill behaviors
  • Game state and difficulty progression management
  • End-of-game scoreboard displaying player name, date, time, and final score

What I Did

  • Designed and implemented core gameplay systems in Java, including combat logic and level progression
  • Developed character selection and sprite management systems
  • Implemented dynamic enemy behaviors with scalable difficulty
  • Built a scoreboard system to record and display player performance data
  • Managed project planning and team coordination using Jira
