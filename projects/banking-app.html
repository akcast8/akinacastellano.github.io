Project Overview

Designed and developed a desktop banking application featuring a graphical user interface and secure database integration to manage financial transactions and user accounts. The system emphasizes role-based access control, data integrity, and efficient financial operations.

Tech Stack

  • Programming Language: Python - Implemented the application logic and integrated the user interface with backend operations.
  • GUI Framework: Tkinter - Designed and built the graphical user interface for Manager, Teller, and Customer roles.
  • Database: PostgreSQL & SQL - Created and managed the relational database for accounts, transactions, and user data.
  • Database Design: Schema & Access Control - Designed role-based access, data integrity, and secure transaction handling.
  • Project Management: Jira - Coordinated tasks, tracked progress, and ensured efficient workflow.

Key Features

  • GUI built with Tkinter for intuitive banking operations
  • Secure PostgreSQL database for structured account and transaction management
  • Core banking functions: deposits, withdrawals, transfers, account creation/deletion
  • Role-based access control for Managers, Tellers, and Customers
  • Structured database schema supporting data integrity and efficiency

What I Did

  • Designed and implemented Tkinter-based GUI for all user roles
  • Built and optimized SQL queries for transaction processing and account management
  • Implemented role-based functionality to enforce access permissions and security
  • Integrated PostgreSQL for persistent data storage
  • Designed database schema to support scalable financial operations
