Physical Diagnosis App (Melanoma Awareness & Screening Tool)

Developed a mobile application in collaboration with an Emory University physician to promote melanoma awareness and support early skin cancer detection through image-based screening and educational resources. The app combines machine learning-powered analysis with guided clinical assessment to help users make informed health decisions.

Tech Stack:

Swift, Xcode, Apple Create ML, Firebase, iOS SDK, Jira

Key Features:

- Machine learning-based skin image scanning that evaluates potential melanoma severity and provides personalized recommendations

- Image upload and scan history tracking to monitor skin changes over time

- ABCDE melanoma assessment checklist integrated with image analysis

- Educational resource hub with medically reviewed melanoma information

- “Find Medical Professionals Nearby” feature to support follow-up care

- Secure, user-specific data storage using Firebase

What I Did:

- Designed and implemented core iOS features in Swift, including the ABCDE Checklist View and scan history tracking system

- Integrated Apple Create ML models to support image-based melanoma screening

- Build Firebase-backed user data storage for scan records and image comparisons

- Researched and incorporated trusted clinical content (e.g., Mayo Clinic guidelines) into the educational section

- Collaborated cross-functionally with a physician and team members using Jira for task management

- Helped present the project at Expo, where it earned 3rd Place for innovation and impact
